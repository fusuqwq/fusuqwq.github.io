python知识点

  • 多行注释

  • print函数

    print(1) print(“helloworld”) print(1,”helloworld”) #数值类型，字符串类型可以直接输出

    !print(1”helloworld”)报错 语法错误

    格式化输出：print('The length of %s is %d' %(s,x))

    ‘The length of %s is %d’ 这部分叫做：格式控制符

    (s,x) 这部分叫做：转换说明符

    % 字符，表示标记转换说明符的开始

    ！单个输出使用%element

    ！不用逗号,进行分隔 直接接上%(s,x)

    python特有元组输出

布尔类型

python中，所有python对象都自动具有bool值，通过内置函数 bool()进行测试

以下所有对象bool值均为False

none false 整形0 浮点数0.0 复数0.0+0.0j 空字符串"" 空列表[] 空元组() 空字典{}

算数运算

除法 java：3/2.0-->1.5 python 3/2-->1.5

整除 java: 3/2 python 3//2

求幂 java: 3^2 python 3**2 ！两**之间不能间隔

python中的单引号和双引号

python中 单引号和双引号包含的字符串本质上完全相同，一般优先选择双引号

str1和str2是没有任何区别的。但是如果遇到需要转义字符的情况，来看单引号和双引号的版本。

单引号需要加 ‘’ 来让编译器判断目前是转义字符，而双引号方便了很多。

反之，如果字符串中有双引号，为了避免使用转义符，可以使用单引号来定义这个字符串。
